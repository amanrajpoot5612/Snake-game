# Information about this game
- Press Enter to start/restart the game
- Use respective arrows keys for navigation
-   "←"   for Left
-   "→"   for Right
-   "↑"   for Top/Up
-   "↓"   for Bottom/Down
- Scores are not permanent, will disappear after restarting the game
